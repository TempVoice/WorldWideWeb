CONFIG

Setup TempVoice

/setup tempvoice

The command /setup tempvoice is used to setup a new creator channel & interface channel with chosen language and color theme.

Setup a Creator Channel | User will be moved

/setup creator (Option I)

The command /setup creator is used to setup a new creator channel for temporary channel creation.

By this method the user has to join the ➕Create Session channel and will be moved to a specially created temporary voice channel afterwards.

Please Keep in Mind:

For the configuration of this creator its important to select the
➕Create Session Channel as Creator for /config commands!

Setup a Creator Channel | User wont be moved

/setup creator (Option II)

The command /setup creator is used to setup a new creator channel for temporary channel creation.

By this method the user must join the ➕Create Session. The ownership of the channel is then overwritten to the user and a new channel is created underneath for the next user.

Please Keep in Mind:

For the configuration of this creator its important to select the
TEMPVOICE CATEGORY Channel as Creator for /config commands!

Setup an Interface Channel

/setup interface

The command /setup interface is used to setup a new interface (channel) for temporary channel management. This interface can be customized by changing title and description. Before creating a new interface the language can be set with /config language and color with /config color.

Change the Default Limit for Temporary Channels

/config limit

The command /config limit allows you to set default user limit for every newly created temporary voice channel.

This default limit can be changed from the user with /voice limit as long the feature is not deactivated with /config feature or /config behavior.

Change the Default Name for Temporary Channels | Fully Changeable

/config name (Option I) 

The name can include following placeholders: {OWNER} {ROLE} {ACTIVITY}

This default name can be fully changed from the user with /voice name as long the feature is not deactivated with /config feature or /config behavior.


🐳・{OWNER}'s Room

Change the Default Name for Temporary Channels | Partly Changeable

/config name (Option II) 

The name can include following placeholders: {OWNER} {ROLE} {CUSTOM}

This default name can be partly changed from the user with /voice name as long the feature is not deactivated with /config feature or /config behavior.


🐳・{CUSTOM}

Please Keep in Mind:

A Partly Changeable Names requires a {CUSTOM} placeholder and cannot include an {ACTIVITY} placeholder. Unless there would be no way for the users to customize the name for their channels!

Change the Default Name for Pre-Created Channels

/config name (Option III) 

This name cannot include any placeholders.

This default name will be replaced when a user joins the channel and gains the ownership.


🐳・Please Join Here

Deactivate or Reactivate Features for Temporary Channels

/config feature

Certain features such changing channel name or limit can be deactivated and reactivated if needed. A deactivated feature wont make it possible to use the corresponding command and the config wont be applied to new temporary channels. This can be useful if you want to force a user limit or privacy mode.

Change the Category for Temporary Channels

/config category

The command /config category allows you to change the category for newly created temporary voice channel.

Please Keep in Mind:

The category cannot be changed for creators which don't move users to their temporary channel because the category is responsible for the configuration of temporary channels.

Default Name Placeholder for Temporary Channels | {CUSTOM}

/config name (option II)

The placeholder {CUSTOM} used in default channel names and will be replaced with the custom channel name of the user who own the temporary voice channel. The user can change this with /voice name. If the user has no custom name it will just display the user name.


🐳・{CUSTOM}

Default Name Placeholder for Temporary Channels | {OWNER}

/config name (option I & II)

The placeholder {OWNER} can be used in default channel names and will be replaced with the display nickname of the user who own the temporary voice channel. With /config behavior you can change it to use the user name instead.


🐳・{OWNER}


Default Name Placeholder for Temporary Channels | {ROLE}

/config name (option I & II)

The placeholder {ROLE} can be used in default channel names and will be replaced with the hoist role name that is displayed in the user list of the user who own the temporary voice channel.


🐍・{ROLE}

Default Name Placeholder for Temporary Channels | {NUMBER}

/config name (option I & II)

The placeholder {NUMBER} can be used in default channel names and will be replaced with the lowest number of temporary voice channels that are active at the moment.


🦑・#{NUMBER}

Default Name Placeholder for Temporary Channels | {ACTIVITY}

/config name (option I)

The placeholder {ACTIVITY} can be used in default channel names and will be replaced with the current users activity. This placeholder can refresh itself with /config behavior when the owner switched the activity.


🎮・{ACTIVITY}
