CONFIG

Setup TempVoice

/setup tempvoice

The command /setup tempvoice is used to setup a new creator channel & interface channel with custom language and color theme.

Setup a New Creator Channel

/setup creator

The command /setup creator is used to setup a new creator channel for temporary channel creation. You can setup up to 15 creator channels with each individual configuration.

Setup a New Interface Channel

/setup interface

The command /setup interface is used to setup a new interface (channel) for temporary channel management. This interface can be customized by changing the language with /config language and color with /config color.

Change the Default Limit

/config limit

The command /config limit allows you to set default user limit for every newly created temporary voice channel.

This default user limit can be changed from the user as long as the feature stays activated. You can deactivate it with /config feature.

Change the Default Name | Fully Changeable

/config name (Option I) 

This default name can be fully changed from the user as long as the feature stays activated. You can deactivate it with /config feature.

Deactivate / Reactivate a Feature

/config feature

Certain features such changing channel name or limit can be deactivated and reactivated if needed. A deactivated feature wont make it possible to use the corresponding command and the config wont be applied to new temporary channels. This can be useful if you want to force a user limit or privacy mode.

Switch the Category

/config category

The command /config category allows you to change the category for every newly created temporary voice channel.

Default Name Placeholder | {CUSTOM}

/config name

The placeholder {CUSTOM} used in default channel names and will be replaced with the custom channel name of the user who own the temporary voice channel. The user can change this with /voice name. If the user has no custom name it will just display the user name.

Default Name Placeholder | {ACTIVITY}

The placeholder {ACTIVITY} can be used in default channel names and will be replaced with the current user activity (game or status). This placeholder can refresh itself with /config behavior when the owner switched the activity.
