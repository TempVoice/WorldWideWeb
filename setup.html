Setup

The following steps will be done is less than a minute. Cool huh?
Join our support Discord server if you need help!

01.

Invite

Invite the Bot to your Discord server here

02.

Permissions

Make sure that the bot gets following permissions:

> Connect
> View Channels
> Manage Roles
> Manage Channels
> Move members
> Use external emojis

03.

Creator Channel

Use the command /creator add to setup a new creator channel

04.

Adjustments

Use the command /creator to customize name, limit...
