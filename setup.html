Setup

The following steps will be done is less than a minute. Cool huh?

1

Invite the TempVoice

Invite the Bot to your Discord server here.
2

Check the permissions

Make sure that the bot gets following permissions: View Channels, Manage Roles, 
Manage Channels, Move members, Connect, Use external emojis.
3

Create a new creation channel

Use the command /creator add to create a new creation channel. Thats it!
4

Make adjustments for your Discord server

Use the command /creator to customize the way how temporary voice channels are created.
