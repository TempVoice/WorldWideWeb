Setup

The following steps will be done is less than a minute. Cool huh?

1

Invite

Invite the Bot to your Discord server here.
2

Permissions

Make sure that the bot gets following permissions: View Channels, Manage Channels,
Move members, Connect, Use external emojis.
3

Commands: /setup

Use the command /setup and choose between the option All channel have the same category and Each channel has its own category.
