/setup

An admin command to setup essential TempVoice elements for temporary voice channel creation and control

tempvoice

Setup TempVoice easly with an assistant

interface

Setup an interface for temporary voice channel control

creator

Setup a new creator channel for temporary voice channel creation or remove an existing one

/voice

A basic command for the channel owner used to change properties

name

Change the name of your temporary voice channel

INFORMATION:
This command can only be used every 5 minutes and this adjustment is saved until the channel is recreated

limit

Change the user limit of your temporary voice channel

INFORMATION:
Limit can be removed if you use 0 has a value and this adjustment is saved until the channel is recreated

privacy

Change the privacy (lock or hide) of your temporary voice channel

INFORMATION:
This adjustment is saved until the channel is recreated

region

Change the region of your temporary voice channel

INFORMATION:
This command can only be used after a regular vote or a valid server wide premium

bitrate

Change the bitrate quality of your temporary voice channel

INFORMATION:
This command can only be used after a regular vote or a valid server wide premium

claim

Claim a temporary voice channel to gain the ownership of it

INFORMATION:
This command can only be used every 10 minutes and can only be used after a regular vote or a valid server wide premium

trust

Trusted users can always join your temporary voice channel

INFORMATION:
The number of trusted users is limited to 25 and added member are saved until the channel is recreated

untrust

Untrusted users are not able to join your locked or hidden temporary voice channel

block

Blocked users are not able to view your temporary voice channel

INFORMATION:
The number of blocked users is limited to 25 and added member are saved until the channel is recreated

unblock

Unblocked users are able to view and join your unlocked or unhidden temporary voice channel

invite

Invited users will get an invite through a dm can join your temporary voice channel

INFORMATION:
This command can only be used after a regular vote or a valid server wide premium and the member will temporary be added to the temporary voice channel

kick

Will kick a user from your temporary voice channel

INFORMATION:
This command can only be used after a regular vote or a valid server wide premium

lists

Shows you a list of all trusted, blocked users and moderators of your temporary voice channel

INFORMATION:
This command can only be used after a regular vote or a valid server wide premium

transfer

Transfer the ownership of your temporary voice channel to another user

INFORMATION:
This command can only be used every 10 minutes and can only be used after a regular vote or a valid server wide premium

thread

Add a temporary thread to your voice channel

/config

An admin command used to configure temporary voice channels

behavior

Change the behavior of certain features and customize them even more

category

Change the category that should be used to arrange temporary voice channels

privacy

Change the default privacy mode for temporary voice channels

name

Set a changeable or unchangeable default name for temporary voice channels

limit

Change the default user limit for temporary voice channels

role

Change the default member role or set a role which is always trusted or blocked

feature

Choose a feature that you want to deactivate or reactivate for your users

color

Change the global theme color of the bot

language

Change the language of the bot messages

censor

Choose which voice channel names are not allowed to use

thread

Change the archive time of temporary threads

placeholder

Change the default text of an empty placeholder
