Setup a creator channel

Used command: /creator channel add
A creator channel is used to create new temporary voice channels. 
Users only have to go into the channel afterwards and will be moved into a newly created voice channel.

PROTIP:
You can remove a creator channel with: /creator remove

Force a specific name

Used command: /creator name
A specific forced name will overwrite all custom user channel names. 

PROTIP:
You can use the placeholder {OWNER} to set the channel owners name user name into the channel name!


Force a specific limit

Used command: /creator limit
A specific forced limit will overwrite all custom user channel limits.

Setup multiple creator channels

Used command: /creator add
Another creator channel allows you to use multiple types of channels.

PROTIP:
You can setup up to 5 creator channels!


Change to advanced mode

Used command: /creator mode
Decide if every temporary voice channel should be in a specific category or if every channel should have its own temporary category.

Change category

Used command: /creator category
Change the the category of a specific creator channel to another one.


Manage commands

Used command: /creator command
Deactivate or reactivate specific commands for your users.
